# 🏦 JMAD Bank Management System

A simple **Java Swing-based Bank Management System** that provides basic banking operations through a graphical user interface. This project demonstrates Java Object-Oriented Programming (OOP), collections, and Swing GUI concepts.

## 📌 Features

### 👨‍💼 Admin

* Secure Admin Login
* Create New Account
* View All Accounts
* Search Account
* Delete Account
* View Total Number of Accounts

### 👤 User

* User Login using Account Number
* Deposit Money
* Withdraw Money
* Transfer Money
* View Account Details
* View Transaction History
* Logout

## 🛠️ Technologies Used

* Java
* Java Swing
* OOP Concepts
* Collections (`ArrayList`)
* UUID (Transaction ID Generation)
* LocalDateTime API

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/JMAD-Bank.git
```

2. Open the project in **VS Code**, **IntelliJ IDEA**, **NetBeans**, or **Eclipse**.

3. Make sure the **Java JDK (17 or above)** is installed.

4. Run the `Mains.java` file.
